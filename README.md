**Éther Counter** é uma ferramenta desenvolvida em C# para auxiliar jogadores de Trading Card Games (TCG). O foco principal é a praticidade e a rapidez no registro de dados durante partidas competitivas ou casais.

---

## 🚀 Funcionalidades

- **Controle de HP Preciso:** Interface numérica para adição e subtração de pontos de vida.
- **Foco em Usabilidade:** Atalhos e botões pensados para quem está com as mãos ocupadas com cartas físicas.

## 🛠️ Tecnologias Utilizadas

- **IDE:** Visual Studio 2022
- **Linguagem:** C#
- **Framework:** .NET
- **Paradigma:** Orientação a Objetos
